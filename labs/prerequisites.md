# Prerequisites

Before you begin deploying Clara, you need to verify that your environment meets all requirements and prepare the necessary tools.


## Required Knowledge

While this lab provides step-by-step instructions, you'll benefit from basic familiarity with:

### Helpful (But Not Required):
- Basic understanding of Microsoft 365 administration
- Familiarity with SharePoint Lists
- Understanding of Copilot Studio concepts
- Basic knowledge of Power Automate
- Experience with Microsoft Entra (Azure AD)

### You'll Learn During the Lab:
- Multi-system agent orchestration
- SharePoint view optimization techniques
- Semantic indexing with Dataverse
- Enterprise agent governance patterns
- Production deployment best practices
