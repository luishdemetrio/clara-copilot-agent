## Lab Overview

The Clara Copilot Agent lab teaches students how to:
- Configure Azure AD applications for OAuth authentication
- Deploy REST APIs to Azure App Service
- Create and configure SharePoint lists
- Import and configure Copilot Studio agents
- Integrate multiple Microsoft 365 services

## Prerequisites

Before starting the lab, ensure you have:
- Active Azure subscription with Owner or Contributor role
- Microsoft 365 tenant with Global Administrator access
- Access to Microsoft Copilot Studio
- Access to Power Automate
- Basic understanding of:
  - Azure Active Directory
  - REST APIs
  - SharePoint Online
  - Microsoft 365 administration

## Lab Structure

Both versions follow the same exercise structure:

1. **Exercise 1**: Configure Azure AD Applications (45 min)
   - Create API app registration
   - Create Copilot Studio app registration
   - Configure permissions and authentication

2. **Exercise 2**: Deploy REST API to Azure (45 min)
   - Create Azure Web App
   - Configure environment variables
   - Deploy code from GitHub

3. **Exercise 3**: Create SharePoint Waitlist (30 min)
   - Create SharePoint list
   - Configure columns
   - Set up data structure

4. **Exercise 4**: Import and Configure Clara Agent (60 min)
   - Import solution to Copilot Studio
   - Configure custom connectors
   - Test integrations

5. **Exercise 5**: End-to-End Testing (30 min)
   - Add test data
   - Verify all integrations
   - Publish the agent
